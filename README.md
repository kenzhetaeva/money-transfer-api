## Quick Start

### Clone repository

```bash
git clone git@github.com:kenzhetaeva/money-transfer-api.git
cd money-transfer-api
```

### Start containers

```bash
make up
```

### Install dependencies

```bash
make composer
```

### Run migrations

```bash
make migrate
```

Application will be available at:

```text
http://localhost:8081
```
